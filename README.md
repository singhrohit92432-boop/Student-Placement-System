# 🎓 Student Placement System

A comprehensive Java-based platform for managing student placements, tracking recruitment drives, and matching candidates with opportunities. Features algorithmic matching, database persistence, and an interactive web dashboard.

## 🚀 Overview

The **Student Placement System** streamlines campus recruitment by managing student profiles, company job postings, application tracking, and placement analytics. Built with Java backend logic and a responsive web frontend, it helps placement officers, students, and recruiters coordinate efficiently.

## ✨ Features

### For Students

- 📝 Register and manage profile (academics, skills, resume)
  
- 🔍 Browse and apply for job openings

- 📊 Track application status

- 📈 View placement statistics


### For Placement Officers

- 👥 Manage student records

- 🏢 Add and manage recruiting companies

- 📅 Schedule campus drives

- 📑 Generate placement reports


### For Recruiters

- 💼 Post job requirements

- 🎯 Filter candidates by criteria (CGPA, skills, branch)

- ⭐ Shortlist and manage applicants


### Core System Features

- 🤖 **Algorithmic Matching** – Match students with jobs based on eligibility

- 💾 **Persistent Storage** – Data stored via database and file system

- 🌐 **Web Dashboard** – Real-time placement activity tracking

- 🐳 **Dockerized** – Easy deployment across environments


## 🛠 Tech Stack

| Layer       | Technology                          |
|-------------|-------------------------------------|
| Backend     | Java              |

| Database    | MySQL / PostgreSQL   |
