About Project

Student Placement System

This project is designed for placement management where students can register, complete their profile, view recommended jobs, apply for jobs and track their applications.

Lanaguages used

  • HTML
  • JAVA
  • MYSQL
  • CSS

Frontend Features

  • Student and company portal
  • Interactive dashboards
  • Job recommendation cards
  • Application tracking
  • Skill-gap analysis
  • Search and sorting

Backend Features

  • Student authentication
  • Company authentication
  • MySQL database integration
  • Job management
  • Application tracking
  • Shortlist and selection system

DAA Algorithms

  • Merge Sort for ranking
  • Binary Search for optimized search
  • Greedy Algorithm for best match
  • Dynamic Programming for skill mapping
